# Optional: contribute your result to an aggregated dataset

The easiest ways to report are the **Validation report** GitHub issue form (the
benchmark script can file it for you) or the NVIDIA forum thread — see the top-level
[README](../README.md).

If you'd also like your raw result included in an aggregated dataset, open a pull
request adding your `validation-report.json` here, named `<gpu-or-spark>-<YYYYMMDD>.json`
(for example `gb10-20260615.json`).

Please remove host names or other personal identifiers before submitting.
